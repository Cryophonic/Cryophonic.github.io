Squid Movies

Squid Movies is an online movie reservation site where users can view and select a couple of movies that they can book tickets for seats in a cinema. This project is created with some of my fellow colleagues using HTML, CSS, and JS.

DM Delights Mobile App

DM Delights mobile app is an online ordering system where customers can order milk tea and other products through online transactions using their mobile devices. The application is created in android studio with cloud technology using google firebase for cloud storage. This project is our capstone project that is currently in development with my fellow programmers.
